# Список майбутніх завдань (Треба зробити)

Цей файл містить список завдань, які ми відклали або плануємо реалізувати в майбутньому, щоб нічого не забути.

## Бекенд та Інтеграції
- [ ] **Тестова оплата:** Провести реальну або тестову транзакцію до самого кінця (списання коштів), щоб перевірити роботу вебхуків та зміну статусу замовлення.
- [ ] **База даних:** Підключити збереження замовлень (наприклад, MongoDB або SQLite) для зберігання історії оплат та клієнтів.
- [ ] **CRM / Telegram-бот:** Налаштувати автоматичну відправку інформації про нові ліди та успішні оплати у Telegram-бот або вашу CRM-систему.
- [ ] **Адаптація під Vercel (Деплой бекенду):** Оскільки фронтенд вже на Vercel, наш Express-сервер (`server.js`) потрібно буде або адаптувати під Vercel Serverless Functions, або розгорнути окремо (наприклад, на Railway, Render чи Google Cloud Run), оскільки Vercel має специфічну архітектуру для Node.js бекендів.
- [ ] **LiqPay:** Повернутися до інтеграції платіжної системи LiqPay, яку ми відклали на майбутні релізи.
